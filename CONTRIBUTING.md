# Contributing

Contributions are welcome! Please read this guide before submitting PRs.

## Development

```bash
# Install dependencies
npm install

# Type check
npm run typecheck

# Run tests
npm test

# Build
npm run build

# Watch mode
npm run dev
```

## Code Style

- TypeScript strict mode
- Run `npm run typecheck` before submitting
- Tests must pass (`npm test`)
- Build must succeed (`npm run build`)

## Pull Requests

1. Fork the repository
2. Create a feature branch (`feat/my-feature`)
3. Make your changes
4. Ensure tests pass and types are clean
5. Submit a PR
